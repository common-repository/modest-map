Self-hosted GDPR compliant map without tracking.

== Description ==

Add a map generated with [MapLibre](https://maplibre.org/) GL JS, an open-source library for publishing maps with customizable styling. Fast displaying of maps is possible thanks to GPU-accelerated vector tile rendering.

= Features =
* Map extracts for any place in the world
* [OpenStreetMap](http://www.openstreetmap.org/) data with 14 zoom levels
* Fully self hosted (<120 MB of data)
* No cookies, no tracking
* No external dependencies
* Define width, height and zoom level for your map
* Mobile-friendly maps
* External links to Google Street View and other providers

= Pro Version (Coming soon) =
* Map extracts with 15 zoom levels
* Routing functionality via external service or self hosted

== Installation ==

Activate the plugin through the 'Plugins' menu in WordPress and click on "Modest Map" in your left navigation menu.

Drag the marker to the desired location and zoom to the initial view level.

Download the map extract for the selected location.

Set additional options for your map (height, width, external link buttons, etc).

Copy and paste the shortcode into your post or page and Modest Map will automatically create the map on your page using the settings you chose above.
